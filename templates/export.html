Інтеграція з іншими системами

Для інтеграції з іншими системами ви можете використовувати наступний код:

Python:
import requests
import json

# Завантаження JSON даних
response = requests.get('http://your-server:5000/export/json')
data = response.json()

# Обробка даних
for post in data:
    print(f"Тип: {post['post_type']}, URL: {post['media_url']}")
JavaScript:
fetch('http://your-server:5000/export/json')
  .then(response => response.json())
  .then(data => {
    // Обробка даних
    data.forEach(post => {
      console.log(`Тип: ${post.post_type}, URL: ${post.media_url}`);
    });
  });
